== NAME

mpkg-deb2tgz - convert debian package to mpkg

== SYNOPSIS

*mpkg-deb2tgz* [OPTIONS] DEBIAN...

== DESCRIPTION

*mpkg-deb2tgz(1)* converts a _debian_ package to *mpkg(1)* format.

== OPTIONS

*--extract-only*::
	Extract only.

*-f or --files*
	Print list of files.

*-h or --help*::
	Display this message.

*-V or --version*::
	Display the version.
